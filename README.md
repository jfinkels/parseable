Dependencies
============

Besides `pdflatex`, compile-time dependencies for the paper include the
following LaTeX packages:

* amsthm.sty
* complexity.sty
* float.sty
* hyperref.sty
* syntax.sty

Compiling
=========

To compile the paper, run 

  ./compile-paper.sh

The output is `parseable.pdf`.
